# Dataset Notes

## PRISM
Use synthetic profile-style samples with hidden ground-truth private attributes.

## MultiPriv
Use image-based privacy perception and reasoning tasks. Do not upload private identity images or restricted dataset files.

## AP²
Use person-level audio samples with real and synthetic labels. Do not upload raw audio if redistribution is not allowed.

## Safety
Remove API keys, private file paths, full raw datasets, and personally identifiable information before publishing.
